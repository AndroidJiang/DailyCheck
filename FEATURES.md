## 🐛 已知限制

### 当前版本限制
1. 删除操作不可撤销
2. 不支持补签过去的日期
3. 图标数量有限（12个）
4. 日历只显示最近30天
5. 无数据备份功能
6. 不支持多设备同步

### 技术限制
1. 使用 Android SDK 28（可能在新设备上有兼容性问题）
2. 本地存储限制（取决于设备空间）
3. 无网络功能（无法云端备份）

## 📝 开发笔记

### 技术选型理由
- **SharedPreferences + Gson**: 简单可靠，适合简单数据存储
- **RecyclerView**: 高性能列表展示
- **Material Design**: 现代化、美观的UI
- **ViewPager + TabLayout**: 成熟的导航方案

### 性能优化
- 使用 ViewHolder 模式减少内存消耗
- 使用 Gson 序列化提高存储效率
- 避免过度绘制（使用 CardView）
- 合理的布局层级（减少嵌套）
