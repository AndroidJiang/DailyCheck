# 快速开始指南

## 🚀 立即开始使用习惯打卡应用

### 第一步：安装应用
1. 在 Android Studio 中打开项目
2. 连接您的 Android 设备或启动模拟器
3. 点击运行按钮 ▶️
4. 等待应用安装完成

### 第二步：创建您的第一个习惯

#### 方法1：从首页创建
1. 打开应用后，您会看到"今日打卡"页面
2. 点击右下角的 **➕ 按钮**
3. 进入创建习惯页面

#### 填写习惯信息
1. **选择图标**
   - 点击顶部的图标区域
   - 从12个预设图标中选择一个
   - 点击确定

2. **输入习惯名称**
   - 在"习惯名称"输入框中输入
   - 例如：喝水、运动、阅读、冥想等

3. **设置每日目标**
   - 使用 ➖ 和 ➕ 按钮调整次数
   - 范围：1-99次
   - 默认：1次

4. **保存习惯**
   - 点击底部的"创建习惯"按钮
   - 返回首页，您会看到新创建的习惯卡片

### 第三步：开始打卡

#### 完成一次打卡
1. 在"今日打卡"页面找到习惯
2. 点击习惯卡片
3. 在弹出的对话框中点击"确认"
4. 看到"✅ 打卡成功！"提示

#### 完成今日目标
- 重复打卡直到达到目标次数
- 完成后卡片会变成绿色
- 显示"🎉 恭喜完成今日目标！继续保持！"

### 第四步：管理您的习惯

#### 查看所有习惯
1. 切换到"习惯管理" Tab
2. 看到所有习惯的列表
3. 显示每个习惯的目标和累计打卡天数

#### 查看打卡日历
1. 在"习惯管理"页面点击任意习惯
2. 查看习惯详情和最近30天的打卡记录
3. 日历说明：
   - 🟢 **绿色** = 已打卡
   - 🟡 **黄色** = 今天
   - ⚪ **灰色** = 未打卡

#### 编辑习惯
1. 在"今日打卡"页面**长按**习惯卡片
2. 修改名称、目标次数或图标
3. 点击"保存"

#### 删除习惯
1. 进入习惯详情页面
2. 滚动到底部
3. 点击"删除习惯"按钮
4. 确认删除

## 💡 使用技巧

### 1. 推荐的习惯设置
- **喝水**：目标 8次/天（每2小时一次）
- **运动**：目标 1次/天（每天运动一次）
- **阅读**：目标 2次/天（早晚各一次）
- **深呼吸**：目标 3次/天（缓解压力）
- **站立活动**：目标 10次/天（久坐办公）

### 2. 养成习惯的建议
- 🎯 **从小目标开始**：先设置1次/天，成功后再增加
- 📅 **保持连续性**：尽量每天都完成，形成习惯
- 🎉 **及时奖励**：完成目标后给自己一个小奖励
- 📊 **定期回顾**：查看打卡日历，看到自己的进步

### 3. 常见问题

**Q: 数据会丢失吗？**
A: 不会。所有数据保存在手机本地，除非卸载应用或清除数据。

**Q: 如何恢复误删的习惯？**
A: 很抱歉，删除操作不可恢复。建议谨慎使用删除功能。

**Q: 能否补签之前的打卡？**
A: 简单版本暂不支持，未来版本会考虑添加此功能。

**Q: 每天什么时候重置？**
A: 第二天首次打开应用时自动重置当日计数。

**Q: 可以添加多少个习惯？**
A: 没有数量限制，但建议不要超过10个，专注于最重要的习惯。
